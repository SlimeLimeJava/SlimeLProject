переменные и вывод текста в консоль

Что такое переменная? 
Переменная - это коробочка, в которой храятся предметы 
определенного типа данных, и хранить можно хранить
только 1 предмет. 

Если сказать более по умному, то переменная это
ячейка памяти комьпьютера которая хранит ваше значение

Что такое тип данных? Воспринимайте тип данных, как тип предмета
которая хранит коробочка, например еду.

В переменную, сразу при создании можно чтото хранить, а можно
оставить коробку пустой, тоесть переменная не будет иметь значения.

в Java существует много типов данных, я перечислю основные, оних 
мы будем говорить в течении этого урока.

Типы данных :

int (Integer) - используется для хранения целых чиспел

float - используется для хранения чисел с плавающей точкой

String - используется для хранения строк (текста)


Поговорим и типе данных int .

Чтобы создать какуюто переменную, мы должны в методе main, написать
одну из следующих конструкций :

Тип_данных имя_переменной;
Тип_данных имя_переменной = значение

В 1 способе, мы создаем переменную, которая не будет иметь 
какое-либо значение, во 2 же способе, у переменной будет
значение.

В названии переменной, не могут быть использованы специалтные 
сииволы, такие как * & < ], но допускается пробел _.
Также в начале названия переменной, нельзя использовать цифры,
используйте их после 1 буквы название. Переменнуб можно
назвать на любом языке, включая Русски1 язык.

давайте создадим переменную, типа данных int : 

public class Main {
    public static void main() {
         int myVar;
         myVar = 7;
    }
}

Выше в методе main, который вам не нежно понимать и запоминать,
так как он сам запомнится, этот метод, обозначает старт программы, 
тоесть что бы там прописали будет запщено при старте.

Тут я создал переменную тип данных int и создал я ее без значения
и ниже я установил через имя переменной, оператора "=" и значения.

Чтобы вывести текст в консоль, мы должны обратиться к классу
System далее к обьекту out и методу print или println.
Что такое классы, обьекты и методы мы разберем позже.
И записфвается так :

System.out.print("Любой текст");

если вам нужно вывести текст в консоль, то вы должны использовать
строки, строка - это текст в ковычках. Если какоето число, то
использовать строки, не обезательно.

Давайте выведем в консоль значение переменной, для этого 
в скобочках пишии название переменной : 

//это все происходит в методе main
int var = 86;
System.out.print(var);

Зачем нужен println, по факту он такойже как и print но, он
с переводом на слелующую строку, тоесть если вы несколько раз 
подряд вызовете print, то все выведется с 1 строку, а если 
println то каждый текст будет писаться со следующей строки.
