Получение данных от пользователя

Для получения данных от пользователя, вы можете 
использовать встроенный класс Scanner. Чтобы начать 
использовать этот класс, необходимо импортировать его в 
ваш проект. Для этого добавьте строку import 
java.util.Scanner; после package com.package;.

Когда вам нужно запросить данные у пользователя, создайте 
объект на основе класса Scanner, указав в параметрах 
System.in. Для подсказки перед созданием объекта, 
используйте метод print, чтобы вывести сообщение, 
например "Введите что-нибудь:"

Например:

package com.example.test;

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        System.out.print("Введите строку: ");
        Scanner myScanner = new Scanner(System.in);
    }
}


Чтобы получить введенные данные, используйте 
соответствующие методы. Например, для получения 
строки используйте nextLine(), для целых чисел - nextInt() 
и т.д.

Давайте напишем программу, которая будет запрашивать у 
пользователя два числа и затем выводить их сумму:

package com.example.test;

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        System.out.print("Введите первое число: ");
        Scanner scanner1 = new Scanner(System.in);
        int firstNumber = scanner1.nextInt();
        
        System.out.print("Введите второе число: ");
        Scanner scanner2 = new Scanner(System.in);
        int secondNumber = scanner2.nextInt();
        
        System.out.print("Сумма введенных чисел: ");
        System.out.print(firstNumber + secondNumber);
    }
}


Для проверки условий на истинность используются условные 
операторы, записывающиеся в следующем формате:

if (условие) {
    // тело условия
}


Также можно добавить блок else, который выполнится, если 
первое условие ложно. Например:

if (условие) {
    // тело условия
} else {
    // тело в случае ложного условия
}


Можно также добавить дополнительные проверки с использованием 
else if:

if (условие) {
    // тело условия
} else if (другое условие) {
    // тело другого условия
} else {
    // тело в случае невыполнения предыдущих условий
}
